# mmd Quick tool

Refactored package source for `mmd Quick tool` based on the latest archived package `version/mmd_quick_tool_1.1.7.zip`.

The generated Serpens implementation is kept in `core.py` to preserve the existing operator IDs, panels, keymaps, and runtime behavior. `__init__.py`, `registration.py`, and `blender_manifest.toml` provide a Blender 4.2+ extension-style package layout matching the structure used by ParamSnap.
